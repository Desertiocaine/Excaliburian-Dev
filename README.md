# EXCALIBURIAN!! Dev Menu

A comprehensive developer menu plugin for the RPG Maker MV game EXCALIBURIAN!!.

## Features

- **F10 Activation**: Press F10 in-game to open the developer menu
- **Equipment Management**: Access legendary weapons and armor including Excaliburn, Excalibur, Mercury Shoes, and more
- **Game State Modifications**: 
  - No-clip mode (walk through walls)
  - Toggle random encounters on/off
  - 10x damage buff for quick testing
- **Safe Integration**: Works seamlessly with the existing game without breaking saves

## Installation

1. Download `DevMenuTest.js`
2. Copy it to your game's `www/js/plugins/` directory
3. Enable the plugin in your plugin manager (see `plugins_example.txt`)
4. Press F10 in-game to access the menu
